Velkommen til Database Preservation Toolkit Enterprise!

En web applikation til at browse og søge i indholdet af flere store arkiverede databaser.

Database Preservation Toolkit Enterprise er en letvægts web viewer til relationelle databaser, nærmere specifikt databaser bevaret i SIARD 2, som bruger SOLR som backend, og som muliggør browsing, søgning og eksport.

Enterprise arkitektur

For store institutioner med mange databaser og brugere

En web applikation som kan skaleres horisontalt til at understøtte mange store databaser, tilgået af hundredevis af bruger på samme tid. Brug for hjælp? Få support.

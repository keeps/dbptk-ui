Bem-vindo ao Database Preservation Toolkit Enterprise!

Um aplicativo web para navegar e pesquisar o conteúdo de multiplas bases de dados arquivadas.

O Database Preservation Toolkit Enterprise é um visualizador Web leve para base de dados relacionais, especialmente se preservada no SIARD 2, usa o SOLR como back-end e permite navegar, pesquisar e exportar.

Arquitetura Empresarial

Para grandes instituições com muitos usuários e bases de dados

Um aplicativo Web que pode ser dimensionado horizontalmente para suportar muitas bases de dados de grande porte, acessados por centenas de usuários simultaneamente. Preciso de ajuda? Obtenha suporte.

Gerenciar várias bases de dados

Sistema único, várias bases de dados

Pesquise nas bases de dados, gerencie seu status, aprimore seus metadados, valide-os e prepare-os para seus usuários finais.

Navegar e pesquisar

Permitir que os usuários acessem o conteúdo da base de dados na Web

Permita pesquisas em um conteúdo preparado, desnormalizado e anonimizado da base de dados

Recurso de exportação

Exportar dados para formato tabular

Permita que os usuários salvem os resultados da pesquisa no Microsoft Excel ou em outro formato de software de planilha para facilmente realizar análises e criar diagramas.

Validação do SIARD

Validar base de dados arquivadas

Valide o SIARD de acordo com as especificações, além de muitas verificações adicionais para obter uma validação completa.

Editar metadados do SIARD

Enriqueça a base de dados arquivada com descrições

Adicione descrições a base de dados, tabelas e colunas para entender melhor seu conteúdo.

Transformação avançada dos dados

Transforme o conteúdo para responder a perguntas úteis

Desnormalização e ocultação de tabela/coluna, para simplificar a navegação/pesquisa e permitir o anonimato do conteúdo.

Login único

Suporte para múltiplos protocolos

LDAP, Active Directory, Database, SAML, ADFS, OAuth2, OpenID, Google, Facebook, Twitter, FIDO U2F, YubiKey, Google Authenticator, Authy, etc. Suporta a definição de autorização interna ou autorização externa configurável.
